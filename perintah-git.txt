#Perintah GIT#

- git init "nama folder(project)" = membuat project baru
- git status = melihat status working folder
- git add = menambah file pada project
- git diff = perbedaan perubahan pada project kita
- git log = melihat history perubahan
- git log --oneline = melihat history perubahan pada satu baris
- git reset = mengembalikan posisi staging ke posisi repo/working folder
